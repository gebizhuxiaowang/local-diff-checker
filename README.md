# 本地文件差异对比工具 Local Diff Checker

纯前端浏览器端多格式内容差异对比工具，**所有文件解析、对比逻辑100%本地执行，不上传任何数据到服务器**，关闭页面后所有临时数据自动销毁，保障隐私安全。

## 功能特性

### P0 已实现核心功能
- ✅ 双栏对比布局（左侧原始内容 / 右侧修改后内容）
- ✅ 两种内容输入方式：手动粘贴/输入 + 本地文件上传
- ✅ 差异高亮展示：
  - 删除内容：红色标记
  - 新增内容：绿色标记
  - 修改内容：黄色标记
- ✅ 差异基础操作：一键对比、清空内容、交换对比内容、差异快速导航（上一处/下一处）
- ✅ 差异统计展示：显示新增、删除、修改的数量
- ✅ 纯本地导出：复制对比结果到剪贴板、导出差异对比文本报告
- ✅ 浅色/深色主题切换
- ✅ 响应式布局，支持PC端优先

### 格式支持
| 格式 | 状态 | 说明 |
|------|------|------|
| 纯文本 | ✅ P0 | 按行对比，显示行号，支持忽略配置（大小写、空格、空行） |
| JSON | ✅ P0 | 自动格式化、语法校验、键值对层级对比 |
| Excel | ✅ P0 | 支持 .xls/.xlsx，按工作表选择对比，单元格级对比 |
| Word | ✅ P1 | 支持 .docx，提取文本后对比 |
| PDF | ✅ P1 | 支持文本型PDF，提取文本后对比 |

### 已知限制
- ❌ 不支持加密PDF / 加密Office文件
- ❌ 不支持图片型扫描PDF（无OCR能力）
- ❌ 旧版 .doc Word 格式支持有限（优先使用 .docx）

## 使用方法

### 直接使用
直接用浏览器打开 `index.html` 文件即可使用，不需要任何服务器、不需要安装依赖。

### 本地部署（静态文件服务）
如果需要通过网络访问，可以使用任意静态文件服务器：

**方法1：Python 简单服务器**
```bash
cd local-diff
python3 -m http.server 8000
```
然后访问 `http://你的服务器IP:8000/index.html`

**方法2：Node.js http-server**
```bash
npm install -g http-server
cd local-diff
http-server -p 8000
```

**方法3：Nginx**
将整个 `local-diff` 文件夹放到 Nginx 的静态资源目录即可。

### 部署到静态网站托管
可以直接部署到：
- GitHub Pages
- Vercel
- Netlify
- 阿里云 OSS
- 腾讯云 COS
- 等任何静态托管服务

## 技术栈
- 原生 HTML + CSS + JavaScript（无框架依赖）
- 第三方开源库（通过CDN引入，不需要本地安装）：
  - [diff](https://github.com/kpdecker/jsdiff) - 文本差异对比
  - [SheetJS (xlsx)](https://sheetjs.com/) - Excel 文件解析
  - [mammoth](https://github.com/mwilliamson/mammoth.js) - Word 文件解析
  - [pdf.js](https://mozilla.github.io/pdf.js/) - PDF 文件解析

## 安全说明
- 无任何接口请求、无数据上报、无埋点
- 不读取本地无关文件，仅解析用户主动上传的文件
- 页面刷新/关闭后，所有解析数据立即释放
- 所有操作都在浏览器本地完成

## 文件大小限制
建议单文件 ≤ 10MB，过大的文件可能导致浏览器卡顿。
